# housing-price-analysis
# Housing Price Analysis using Linear Regression

## Overview

This project analyzes housing prices using linear regression techniques on the Kaggle House Prices dataset.

The goal is to understand which factors most strongly influence house prices.

Key variables analyzed:
- Living area
- Overall quality
- Garage capacity

## Business Questions

1. Does a larger house always cost more?
2. Does quality matter more than size?
3. What role does garage size play in pricing?

## Dataset

Dataset source:
https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Methods

- Exploratory Data Analysis (EDA)
- Data Cleaning
- Linear Regression
- Multiple Regression

## Key Findings

- Living area positively affects price
- Overall quality is the strongest predictor
- Garage size has moderate influence
